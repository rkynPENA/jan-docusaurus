This guide will help you to use the desired AI models, setting up the threads, and chat with the model.

### Step 1: Using The AI Model

Before using Jan, it is necessary to select the model that are supported by the program.

Each models have their own purposes, capabilities, and different requrements. You can choose which model that can fulfil your requirements based on their proficiency.

For more info, go to [list of supported models](/docs/models-list/). 

### Step 2: Setting Up a Thread

After set the model, it is necessary to set your thread before chat with the model.

There are few settings you can consider, such as title for the thread, set the assistant instructions, and set the type of model along with the API key.

### Step 3: Chat with the model

After setting up the model and the thread, you can now chat with the model and use them based on your requirements.
