### Step 1: Installing Jan

To fully experience Jan, you need to install the program.

Firstly, go to [jan.ai](https://jan.ai/), select your desktop operating system (**MacOS**, **Windows**, or **Linux**), and install the program.

To learn more about system requirements for your operating system, go to [Installation guide](https://visionary-cajeta-bf2527.netlify.app/docs/install).

### Step 2: Select AI Model

Before using Jan, it is necessary to select the model that are supported by the program.

Each model has their purposes, capabilities, and different requirements. You can choose which model can fulfill your requirements based on their proficiency.

To select AI models, go to the Hub, and from there, you can select the models that you would like to use.

For more info, go to [list of supported models](/docs/models-list/). 

### Step 3: Use the AI Model

After you set up your model and the program, you can chat with the model and use them based on your requirements.
